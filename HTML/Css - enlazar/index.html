hay 3 maneras de implementar el estilo CSS a nuestros documentos HTML, 
        todas son igual de eficientes solo que algunas son más cómodas que otras

        
 Mediante la vinculación con archivos externos

        La vinculación con archivos CSS externos, se hacen dentro de las etiquetas
        head y con el elemento link del HTML.
    
      Mediante las etiquetas style
        
 Mediante las etiquetas style dentro del documento

        La segunda opción sería mediante las etiquetas style tambien 
        dentro de las etiquetas HEAD
    
      OTRO METODO Como valor del atributo “style”
      
        
 Como valor del atributo “style” en el elemento del HTML

        En HTML, la mayoría de etiquetas como son: h1, h2, h3, p, div, span, table, body,
        input, entre otras, tienen diversos atributos entre los que existe uno que 
        se llama style = “”, y dentro de las comillas dobles irá todo el código CSS.
    



      Notese que en esta pagina coexisten los tres metodos para 
      aplicar los estilos, y notese que el atributo color a los 
      elementos h2 se le esta dando valor usando los tres metodos,
      sin embargo el color azul que se le aplica con la etiqueta style,
      no es tomado en cuenta y se le aplica el color negro especificado
      en el metodo "link" a traves de la hoja de estilo; y el valor rojo dado
      con el atributo style dentro del ultimo h2 se aplica con preferencia
      de los otros dos metodos.
      Por lo tanto existe un orden de preferencia al aplicar los estilos
      segun el metod utilizado, siendo el prioritario el atributo style
      dentro de la etiqueta de apertura. Sin embargo lo recomendado es
      utilizar solo un metodo "link" a traves de la hoja de estilo.
