Jerarquía de importancia para resolver conflictos entre varias declaraciones de estilos

COMO REGLA: formas de declaración más generales --> menos respetadas en caso de conflicto

  1. Declaración de estilos con fichero externo. (Para todo un sitio web)
  2. Declaración de estilos para toda la página. (Con la etiqueta STYLE en la cabecera de la página)
  3. Definidos en una etiqueta en concreto. (Utilizando el atributo style en la etiqueta en cuestión) 
        SINTAXIS DE UNA REGLA
    
                    selector {

                        atributo: valor;
                        atributo: valor;
                        atributo: valor;

                    }
